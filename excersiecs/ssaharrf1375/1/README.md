```ruby
w=ones(400);
```

 <div dir="rtl">
ایجاد دو حلقه تو در تو برای حرکت عمودی و افقی و قرار دادن خانه ها به صورت یکی در میان به رنگ مشکی
 </div>

```ruby
for i=0:40:400-40     
    for j=0:40:400-40         
        w(i+1:i+20,j+1:j+20)=0;         
        w(i+20:i+40,j+20:j+40)=0;     
    end
end
```
 <div dir="rtl">

 </div>
